Technology
--------------------------------------------
- Java 17+
- Javalin
- PostgreSQL
- Maven
- JDBC
- HTML
- CSS

Prerequisites
-------------------------------------------
To build and run this project, you will need:

- Java 17+
- Maven
- PostgreSQL
- PG4Admin

Installation
-----------------------------------------------
- Clone the project repository to your local machine.
- Create a database Translation and specify your connection parameters in the configuration.properties file, also add Deepl API key .
- Install maven (mvn clean install)

Start
---------------------------------------------------
- Start Server
- Open the application in any browser with http://localhost:7000/
